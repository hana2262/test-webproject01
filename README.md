# 🌤️ Weather Checker

A simple and beautiful weather查询小工具 built with HTML, CSS, and JavaScript.

## Features

- 🔍 Search weather by city name
- 🌡️ Display current temperature
- 💧 Show humidity and wind speed
- 🎨 Beautiful gradient UI design
- 📱 Responsive for mobile and desktop

## Tech Stack

- **HTML5** - Page structure
- **CSS3** - Styling with gradient background
- **JavaScript** - Dynamic weather fetching
- **Open-Meteo API** - Free weather data (no API key required)

## How to Use

1. Open `index.html` in your browser
2. Enter a city name (e.g., "Shanghai", "Beijing", "Tokyo")
3. Click "Search" or press Enter
4. View the weather information!

## API

Uses [Open-Meteo](https://open-meteo.com/) - A free, open-source weather API that requires no authentication.
