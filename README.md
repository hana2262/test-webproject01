# ☁️ Weather Quotes

A beautiful weather checker with inspiring quotes, built with HTML, CSS, and JavaScript.

## ✨ New Feature

Now shows **inspirational quotes** based on the weather! 
- Sunny days bring motivation ☀️
- Rainy days remind you of hope 🌧️
- Every weather has its own beauty 💫

## Features

- 🔍 Search weather by city name
- 🌡️ Display current temperature
- 💧 Show humidity and wind speed
- 💬 Weather-based inspirational quotes
- 🎨 Beautiful gradient UI design
- 📱 Responsive for mobile and desktop

## Tech Stack

- **HTML5** - Page structure
- **CSS3** - Modern styling with gradient background
- **JavaScript** - Dynamic weather fetching
- **Open-Meteo API** - Free weather data (no API key required)

## How to Use

1. Open `index.html` in your browser
2. Enter a city name (e.g., "Shanghai", "Tokyo", "New York")
3. Click "Search" or press Enter
4. View the weather and read your daily inspiration! ✨
